CSS

Framework compass
=================

Instalación
~~~~~~~~~~~

.. code-block:: bash 

    sudo apt-get install rubygems
    gem install compass
    gem install sass

Compilar un fichero
~~~~~~~~~~~~~~~~~~~

.. code-block:: bash 

    compass compile scss/srceen.scss

Para compilar en función de los cambios que se realicen en un directorio
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

.. code-block:: bash 

    compass watch
